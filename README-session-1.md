# Session 1 

In this session we did a small JS & Ts recap and kickstarted react with a funny cat project

## Overview
A focused introduction to modern front‑end fundamentals: JavaScript core concepts, basic TypeScript benefits, and a practical React setup using Vite.

## Topics Covered
- **JavaScript Fundamentals**
    - Functions & modules
    - Higher‑order functions (map, filter)
- **TypeScript Introduction**
    - Advantages of type safety and early error detection
- **React Hands‑On**
    - Created a Vite project scaffold
    - Built React components and passed props
    - Implemented dynamic data flow between components

## Run the React app

To run the example React app for Session 1, from the repository root run:

```fish
cd react-cats-app
pnpm install
pnpm run dev
```
